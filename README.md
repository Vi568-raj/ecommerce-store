# 🛒 UrbanCart - Full Stack E-Commerce Store

UrbanCart is a complete full-stack e-commerce web application built using **Node.js, Express.js, MongoDB, and Vanilla JavaScript**. It provides a modern online shopping experience with user authentication, product management, shopping cart, checkout system, order tracking, and an admin dashboard.

This project is designed for learning, portfolio demonstration, and understanding how a complete e-commerce platform works from frontend to backend.

---

## ✨ Features

### 👤 User Features
- User registration and login
- Secure authentication using JWT
- Browse available products
- Search and filter products
- View product details
- Add products to cart
- Update cart quantity
- Checkout process
- View order history
- Manage user sessions

### 🛍️ Shopping Features
- Product listing
- Product filtering
- Shopping cart management
- Order placement
- Order tracking
- Responsive storefront design

### 🔐 Admin Features
- Admin authentication
- Admin dashboard
- Add new products
- Update product details
- Delete products
- Manage customer orders
- View order information
